This is the readme.txt file for the models associated with the 
papers shown below:

1) Baeza-Loya, S and Eatock, RA. "Effects of transient, persistent, and resurgent sodium currents on excitability and spike regularity in vestibular ganglion neurons". 

To run the simulation type Single_Compartment_Annotated at the Matlab command line.

To control the simulation parameters, open and edit the Single_Compartment_Annotated file.

Original code graciously shared by R. Kalluri and can be found in location indicated in Ventura and Kalluri, 2018 and Hight and Kalluri, 2016.
